Project Summary


The purpose of this project was to analyze how weather changes as we get closer to the equator. This analysis involved creating a random list of cities alongisde their latitude and longitude. From there, an Open Weather API pull was used to gather weather data for each of the 500+ cities as of April 30, 2021.

After compiling the dataset, Matplotlib was used to plot key relationships between different weather variables and their latitudes. This site includes the source data and visualizations created as part of the analysis. Explanations and descriptions of trends are also included.
